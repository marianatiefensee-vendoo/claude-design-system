Fields

The Field family: single-line text (Field), dropdown trigger (FieldSelect), multi-line auto-growing text (FieldTextArea), and chip-input with optional suggestions (FieldStyleTags). All share the Title row (label + optional Tag + AssistiveChip) and the Supporting text row. Inline chips use --sys-secondary-container / --sys-on-secondary-container (deliberate override of Figma's --sys-primary-fixed for system-wide consistency across Tag, StyleTag, and suggestions).
