Fields

Single-line text input (Field) and dropdown trigger (FieldSelect). 56 dp tall, 4 dp corner, 1 dp border that shifts color — --sys-outline at rest, --sys-primary on focus or just-entered (Filled), --sys-outline again once validated or when content overflows, --sys-error in error. Label row supports an optional Tag chip and an AssistiveChip (right-aligned). Helper text below with a 16 dp left gutter so it aligns to the input text, not the border. Matches Figma 1243:5664 (Field) and 1243:5824 (Field/Select).

Live sandbox — configure every prop and watch the field react. Toggle Tweaks from the toolbar to open the control panel.
